# Community

Welcome to the Web Context Protocol community! We're building the future of AI-web interaction together, and we'd love for you to be part of it.

## Join Our Discord

The best way to connect with the WCP community is through our Discord server. Here you can:

- **Discuss ideas** and get feedback on your implementations
- **Get help** with technical questions and troubleshooting
- **Share your projects** and see what others are building
- **Contribute** to the protocol development
- **Stay updated** on the latest news and releases

## Contributing to WCP

We welcome contributions from developers of all skill levels! Here's how you can get involved:

### Ways to Contribute

<AccordionGroup>
  <Accordion title="Code Contributions">
    - **Protocol Implementation**: Help implement WCP in different languages and frameworks
    - **Tooling**: Build developer tools, validators, and testing utilities
    - **Examples**: Create example implementations and use cases
    - **Documentation**: Improve docs, add tutorials, and fix typos
  </Accordion>

  <Accordion title="Community Building">
    - **Share Your Projects**: Show off what you've built with WCP
    - **Help Others**: Answer questions in Discord and GitHub discussions
    - **Content Creation**: Write blog posts, tutorials, or create videos
    - **Feedback**: Test new features and provide constructive feedback
  </Accordion>

  <Accordion title="Protocol Development">
    - **Specification**: Help refine and expand the WCP specification
    - **Testing**: Test implementations and report bugs
    - **Research**: Explore new use cases and technical approaches
    - **Standards**: Contribute to related web standards and best practices
  </Accordion>
</AccordionGroup>

### Getting Started

1. **Join Discord**: Start by joining our [Discord community](https://discord.gg/cGfZmVA7) to introduce yourself
2. **Explore the Code**: Check out our [GitHub repository](https://github.com/webcontextprotocol) to see what's being worked on
3. **Pick an Issue**: Look for issues labeled `good first issue` or `help wanted`
4. **Start Small**: Begin with documentation improvements or small bug fixes
5. **Ask Questions**: Don't hesitate to ask questions in Discord - we're here to help!

### Development Guidelines

When contributing code:

- Follow the existing code style and conventions
- Write clear commit messages
- Add tests for new functionality
- Update documentation as needed
- Submit pull requests with clear descriptions

### Code of Conduct

We are committed to providing a welcoming and inclusive environment for everyone. Please:

- Be respectful and constructive in all interactions
- Help create a safe space for learning and collaboration
- Report any inappropriate behavior to the community moderators
- Follow our [Code of Conduct](https://github.com/webcontextprotocol/.github/blob/main/CODE_OF_CONDUCT.md)
